# HangmanGame
Python Hangman Game
<br><br>
Creator: [IgnacioPrados](https://github.com/IgnacioPrados)
<br><br>
Key Technologies used: Python

## Run the app
Using [Python](https://www.python.org/)
```
python hangman.py
```

## Usage
- Guess the word given letter by letter. If you run out of opportunities you lose.
